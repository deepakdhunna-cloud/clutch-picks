## Screenshot Requirements

### iPhone Screenshots (Required)
You need screenshots for these sizes. Take screenshots from the app showing key features.

#### 6.7" Display (iPhone 15 Pro Max) - REQUIRED
- Resolution: 1290 x 2796 pixels
- Required: 1-10 screenshots

#### 6.5" Display (iPhone 14 Plus) - REQUIRED
- Resolution: 1284 x 2778 pixels
- Required: 1-10 screenshots

#### 5.5" Display (iPhone 8 Plus) - Optional but recommended
- Resolution: 1242 x 2208 pixels

### iPad Screenshots (Required if supporting iPad)
Since your app supports tablet (`supportsTablet: true`), you need iPad screenshots.

#### 12.9" Display (iPad Pro 6th Gen) - REQUIRED
- Resolution: 2048 x 2732 pixels
- Required: 1-10 screenshots

#### 12.9" Display (iPad Pro 2nd Gen) - REQUIRED
- Resolution: 2048 x 2732 pixels

### Recommended Screenshots to Take

1. **Home Screen** - Show the Clutch Picks branding and today's games
2. **Clutch Picks** - Show a Clutch Pick card with the confidence rating
3. **Sports Browse** - Show the sports categories (NFL, NBA, etc.)
4. **Game Detail** - Show a specific game with teams and prediction
5. **Live Games** - Show live game tracking with real-time scores
6. **Profile/Stats** - Show user profile with pick statistics
7. **Social/Following** - Show the community features

### Screenshot Tips
- Use clean, compelling content (good matchups, realistic data)
- Consider adding marketing text overlays in Figma/Photoshop
- Show the app in action, not just static screens
- Highlight key features in each screenshot
